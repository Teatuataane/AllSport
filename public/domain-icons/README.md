# Domain icons — Canva export convention

Drop one PNG per domain in this folder and it appears automatically on the Personal Bests page (`/prs`) collapsible domain headers, and anywhere else `DomainIcon` is used. No code change needed.

**Export rules:**
- Transparent background, solid single-colour silhouette (black is fine — the app recolours it to the domain colour via CSS mask, so it will show correctly on the dark theme)
- Square-ish crop, subject filling most of the canvas
- ~1000×1000 px (matches the event-icon set)
- Filename = domain slug exactly (see list below)
- Until a domain has an icon, the app shows the domain number in the tinted tile as a fallback
